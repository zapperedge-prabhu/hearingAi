## SECTION 2: TYPES OF EVENTS LOGGED (2:00 - 4:15)

[VISUAL: Show different event categories]

**NARRATOR:**

"Zapper logs events across all major system activities:

**File Management events:**
- File uploaded — User uploaded a file. Size, location, file type
- File downloaded — User downloaded a file. Using web, API, or which app
- File deleted — User deleted a file. Why, if recorded
- File preview — File viewed but not downloaded

**User Management events:**
- User Add — New user added to system
- User Update — User modified
- User disabled — Account locked
- User enabled — Account reactivated
- User deleted — Account removed 

**Organization Management events:**
- Organization created — New organization provisioned
- Organization settings modified — Name, description
- Organization deleted — Organization deleted


Every event is logged automatically. You don't have to enable individual event types. Everything is recorded by default."

## SECTION 3: SEARCHING & FILTERING LOGS (4:15 - 6:30)

[VISUAL: Show the search and filter interface]

**NARRATOR:**

"With thousands of events per day, you need powerful search and filtering.

Zapper provides multiple ways to find specific events:

**Filter by user:**
'Show me all actions performed by john.smith@acme.com.' Helpful when investigating what a specific person did.

**Filter by action type:**
'Show me all file downloads.' Or 'Show me all failed access attempts.' Focus on the event type you care about.


**Free-text search:**
Type any text. Zapper searches file names, user names, descriptions. 'Search for Smith' finds all events involving anyone named Smith or any files containing Smith.
